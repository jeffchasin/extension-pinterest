Pinterest Conversion Tracking

Use the Pinterest tag on your website to build audiences based on actions they take, and gather valuable insights. Learn more about the Pinterest Tag for businesses and websites.

Note

The Pinterest base code (the Pinterest tag) is required on pages where you will track events. This extension provides that base code.

You can add a Pinterest Load Base Code action to any global rule that uses a Page Top event type to ensure the base code will be available for Pinterest event actions.

To find your Pinterest Tag ID, sign in to your Pinterest Ads account, click on Ads, then Conversions.
